# Brickmatica
A port of backported lunatrius' schematica from ModLoader to Babric.

Screenshot taken on [2beta2t](https://2beta2t.net/)
![Screenshot](img/screenshot.png)

### Requirements
- [Babric](https://babric.github.io/)
- No StAPI
